

## Doorium Service — Главная страница (Hero-блок + Шапка)

### Цветовая система
Создаём систему на основе присланных палитр — брутальный микс тёмных, серых и тёплых бежево-золотых тонов:
- **Основной фон**: тёмный (#0A0908 / #181712) — для hero, хедера
- **Светлые секции**: бежевый (#EAE0D5 / #E9DACC) — для контентных блоков
- **Акцент**: золотой (#9B7B3F / #C6AC8F) — кнопки, выделения
- **Серые тона**: (#686A6A / #968E8B) — вторичный текст, бордеры
- **Текст на тёмном**: светлый (#E4E4E5 / #BABABB)

### Шапка (Header)
- Фиксированная прозрачная шапка поверх hero
- Логотип Doorium Service слева (загруженное изображение)
- Навигация: Услуги, Наши работы, Новости, Контакты
- Кнопка «Заявка» с золотым акцентом справа
- Бургер-меню на мобильных

### Hero-блок
- Полноэкранное фото-заглушка (placeholder с тёмным оверлеем — позже замените на реальное фото дверей/интерьера)
- Крупный заголовок «ПРЕМИАЛЬНЫЙ МОНТАЖ» — большие буквы, tracking wide, брутальный шрифт
- Подзаголовок: краткое описание услуг Doorium
- CTA-кнопка «Оставить заявку» в золотом цвете

### Общий стиль
- Современный, брутальный: крупная типографика, контрастные блоки, минимализм
- Плавные анимации при скролле (fade-in)
- Полностью адаптивный дизайн

### Структура страниц (как PrimeDoor, делаем поэтапно)
1. **Главная** ← начинаем с неё
2. Услуги (межкомнатные / входные двери)
3. Наши работы (портфолио)
4. Новости / Статьи
5. Контакты
6. Политика конфиденциальности

Сейчас реализуем только **шапку + hero-блок + цветовую тему**, а остальные секции будем добавлять по одной.
